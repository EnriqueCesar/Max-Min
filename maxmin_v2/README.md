# Max & Min Centro Norte 2.0

Herramienta estática lista para GitHub Pages.

## Vistas
- **Max&Min:** foto + marcadores + mínimo/máximo por artículo.
- **Consulta:** validación de listas con filtros, categoría y selección múltiple.
- **Acomodo:** foto + marcadores + nombres editables y mínimos/máximos opcionales.

## Fórmulas
- Uso promedio = promedio de la columna H de las semanas seleccionadas.
- Uso mínimo = uso promedio / 7.
- Uso máximo = uso mínimo × factor de pedidos: 2=×5, 3=×4, 4=×3, 5=×2.
- Unidad/Pick Pack se toma de la pestaña `Unidad_PickPack`.
